Guide:
In order to print a list of all books whose titles contain a certain string S, use python3 books.txt --book S
In order to print a list of all athors whose names contain a certain string S, and for each of those authors print a list of their books, use python3 books.txt --author S
In order to print a list of book published between years A and B (inclusive) use python3 books.txt --year1 A --year2 B
